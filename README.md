# FoodTrends-Understanding-Customer-Preferences-in-FB-Project


An interactive Power BI dashboard designed to analyze food consumption patterns, nutritional balance, and lifestyle impact—helping uncover actionable insights about customer dietary behavior and health trends.

**1. Short Description / Purpose**
 
This project is a comprehensive data visualization and analytics solution built using Power BI to explore how dietary preferences, lifestyle habits, and nutritional intake influence overall health.
It enables users to identify trends, detect imbalances, and make data-driven decisions related to food and wellness.

**2. Data Source**
   
The dataset used in this project includes structured information related to:

• Food items and meal types (Breakfast, Lunch, Dinner, etc.)

• Nutritional values (Calories, Protein, Carbohydrates, Fat, Fiber)

• User attributes (Age, Gender, Weight, Activity Level)

• Dietary preferences (Omnivore, Vegetarian, Vegan, Pescatarian)

• Health indicators (BMI, Disease Types)

The data was preprocessed and transformed using Power Query to ensure accuracy and consistency.

  Link for the datasets:

   1. https://www.kaggle.com/datasets/adilshamim8/daily-food-and-nutrition-dataset/data
   
   2. https://www.kaggle.com/datasets/bitanianielsen/nutrition-daily-meals-in-diseases-cases

**3. Tech Stack**
   
• Power BI Desktop – Dashboard development and visualization

• Power Query – Data cleaning and transformation

• DAX (Data Analysis Expressions) – Measures, KPIs, and calculated logic

• Data Modeling – Relationships and schema design

• Excel Dataset – Source data

**5. Features / Highlights**

• Interactive and dynamic dashboards

• KPI-driven design (Calories, BMI, Nutrient Intake)

• Advanced DAX calculations (Calorie Status, Ratios, Efficiency Metrics)

• Multi-dimensional filtering (Diet, Activity, Gender, Disease)

• Clean and consistent UI/UX design

• Real-world analytical approach (Problem → KPI → Insight → Action)

**6. Key Insights**

• Higher activity levels correlate with higher calorie intake but better BMI control

• A large percentage of individuals are above their calorie target

• Dietary preferences influence macronutrient distribution significantly

• Breakfast plays a crucial role in daily nutritional balance

• Poor nutritional balance is linked to higher health risk categories

**7. Example Use Case**

**Business Question:**

Are individuals maintaining a healthy calorie intake based on their lifestyle?

**Analysis Approach:**

• Compare calorie intake across activity levels

• Evaluate BMI trends

• Segment users by calorie status (Within vs Above target)

**Insight:**

Sedentary individuals tend to exceed calorie targets more frequently, leading to higher BMI levels.

**Actionable Recommendation:**

• Introduce personalized meal plans

• Encourage activity-based calorie targets

• Promote balanced macronutrient intake

**8. Conclusion**

This project demonstrates that data analytics is not just about tracking consumption trends — it is about translating customer behavior into actionable strategies that improve both business outcomes and consumer well-being.
